    select in element¶

    当弹出层中有 select 时，IE 中在 select 选项上移动时会意外隐藏浮层。

    进行了修复，统一使所有浏览器下在 select 选项上移出时始终不触发隐藏行为。

    在实践中，请尽量保证 select 的选项层始终包含在浮层中。
