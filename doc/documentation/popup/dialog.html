Dialog

基础对话框组件，提供对话框显示隐藏、dom 结构自定义、定位、确定取消关闭等功能特性。

    Dialog 继承了 Popup，可使用其中属性和方法。

    配置说明¶

    title string¶

    对话框标题。

    button array¶

    按钮组，格式：

    [{
    role : 'confirm',
    cls : 'ui-dialog-button-orange',
    text : '确定',
    autofocus: true
}, {
    role : 'cancel',
    cls : 'ui-dialog-button-white',
    text : '取消'
}]

    classPrefix string¶

    统一样式前缀。

    skin string¶

    对话框主题。

    content element|string¶

    指定内容元素，可以是 iframe 地址，默认为 element。

    • 指定为 iframe：iframe:http://www.baidu.com
    • 指定为 ajax：ajax:http://www.xxxx.com/partial.html

    closeTpl string¶

    关闭按钮可以自定义。

    initialHeight number¶

    iframe,ajax 类型时，dialog 的最初高度。

    initialWidth number¶

    iframe,ajax 类型时，dialog 的最初宽度。

    autoFit boolean¶

    iframe 类型时是否自适应。

    静态方法¶

    alert¶

    var Dialog = FNX.include('popup/dialog');

Dialog.alert('信息标题', '信息内容', function () {
    console.log('callback')
});

    confirm¶

    var Dialog = FNX.include('popup/dialog');

Dialog.confirm('信息标题', '信息内容', function (flag) {
    console.log(flag)
});

    show¶

    var Dialog = FNX.include('popup/dialog');

Dialog.show('信息内容');
