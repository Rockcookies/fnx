Popup 示例

Popup 基本示例。

    1. 基本调用，内容传入字符串¶

    var Popup = FNX.include('popup/popup');

var popup = new Popup({
    template: '<div class="popup">我是内容啊！！！</div>',
    width: 500,
    height: 200,
    zIndex: 99,
    style: {
        background: 'yellow',
        color: 'green'
    }
});

$('#example1').click(function () {
    popup.show();
});

    2. 内容可传入 DOM 对象¶

    传入了DOM
    var Popup = FNX.include('popup/popup');

var popup = new Popup({
    element: '#example2-dom'
});

$('#example2').click(function () {
    popup.show();
});
